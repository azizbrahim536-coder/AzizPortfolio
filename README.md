# AzizPortfolio

AzizPortfolio is a personal portfolio website built with Angular.  
It showcases my skills, projects, and learning journey as a junior full-stack developer.

## Description

This portfolio presents the main projects I have built using Angular, Spring Boot, MySQL, Flask, and AI integrations.  
It includes project cards with descriptions, technologies used, screenshots, and GitHub links.
